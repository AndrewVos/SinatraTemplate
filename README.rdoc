= Sinatra Template

== Installation
Open up a terminal.
	git clone git@github.com:AndrewVos/SinatraTemplate.git
	mv SinatraTemplate <application_name>
	cd <application name>

== Rackup
To run the site open up a terminal and run "rackup" in your application directory. Your site will now be up and running at http://localhost:9292

== Testing
To run rspec tests just run "autospec" from your application directory.
